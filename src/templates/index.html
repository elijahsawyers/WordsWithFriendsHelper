Instructions

To fill out the game board, click on a cell, and type the key that you would like to place into the selected cell.

Next, input your current seven letters into the cells on the right-hand side of the screen the same way that you populate the game board cells.

Lastly, press the "Go!" button, and the best possible move, given your seven letters and the current state of the game board, will be displayed on the gameboard with letters with a purple border.

Note: to input a wildcard character, use the space-bar. To delete a character, select the cell and press backspace. To deselect a cell, press escape.
